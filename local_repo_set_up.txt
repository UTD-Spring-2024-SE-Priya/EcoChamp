1) To create virtual environment in your project directory:
py -3 -m venv .venv
.venv\scripts\activate

2) Select .venv interpreter via Command Palette (Ctrl+Shift+P)

3) To create local repository in project directory:
git init
git remote add origin https://github.com/UTD-Spring-2024-SE-Priya/EcoChamp.git
git fetch origin
